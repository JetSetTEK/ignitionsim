## Community pattern notes (paraphrased in our voice)
- Near-unanimous VR consensus: a Stream Deck is useless in a headset; buy a physical box. Monitor racers rave about labels + SimHub telemetry keys.
- Endurance/iRacing crowd: once you manage pit functions + black boxes, some extra control surface stops being optional.
- Under-discussed gotcha: SimHub and the Stream Deck app fighting over the same vJOY device is the usual cause of "my buttons randomly stopped working."

## Credible warnings / caveats
- Binding limits: some sims cap registered controller buttons / keyboard binds; 32 keys != 32 usable in-game slots everywhere.
- Mounting: loose deck slides mid-race; unbolted box walks under a hard toggle flick. Budget a Race Deck case or VESA/RAM mount.
- Software dependency: Stream Deck needs its app running; plugins (context labels, SimHub) can break on updates. A dumb box just presents as a controller.
- VR is a hard line — do not buy a deck "for now" if you race in a headset.
