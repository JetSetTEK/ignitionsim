## Expansion priority — biggest gaps × hottest demand

Across bays, the SAME missing ecosystem keeps surfacing as the community’s highest-ROI upgrades. Lead the catalog expansion here:
1. **Tactile / bass shakers + amps** (Dayton BST-1, NLR HF8, ButtKicker, Fosi amps) — #1 recommended immersion upgrade in every bay; cheap, high attach-rate, high-affiliate.
2. **Seating / cockpit rigs** — catalog has ~0 seats; universal need (GT buckets, 80/20 rigs, foldable cockpits).
3. **VR headsets** (Quest 3/3S, Pimax Crystal Super, Bigscreen Beyond 2) — THE immersion story of 2026; applies to every bay.
4. **Monitors + stands** (49" OLED G9, 57" Neo G9, triple stands) — display is a whole missing category.
5. **PC / GPU** (RTX 5080/5090) — the catalog ignores compute, yet it gates VR/triples.
6. **Motion + wind** (NLR Motion V3, DOF Reality, wind sims) — big-ticket halo + honest editorial angle.
7. **Audio, ambient/bias lighting, cable management, powered USB hubs, tablet/EFB mounts** — cheap, high-satisfaction “complete-your-build” attach cluster.
